# Node JS Basic: displayMessage

This module exposes `displayMessage(message)` which prints the provided string to STDOUT.

## Files
- 0-console.js: Exports `displayMessage`.
- 0-main.js: Sample runner that calls `displayMessage('Hello NodeJS!')`.

## Usage
From the repository root:

```powershell
node Node_JS_basic/0-main.js
```

Or from inside the Node_JS_basic folder:

```powershell
cd Node_JS_basic
node 0-main.js
```

Expected output:

```
Hello NodeJS!
```
